Fencing Tracker App

A web application for parents of youth fencers to track tournament results

Test the App

Follow these steps to run the Fencing Tracker App on your local machine:

  1. Clone the Repository

    First, clone the repository to your local machine:

    git clone https://github.com/yourusername/fencing-tracker.git
cd fencing-tracker

  2. Install Dependencies

    Make sure you have Node.js and npm installed, then run:

    cd app
npm install

  3. Start the Development Server

    Start the React development server:

    npm start

    This will open the app in your default browser at http://localhost:3000.

  4. Log In

    On the login page, you can use any email and password (for the demo version) or click the "Demo Login" button.

  5. Add Fencers

    Navigate to the "Fencers" section and add your fencers.

  6. Add Tournaments

    Go to the "Tournaments" section and add tournaments.

  7. Record Results

    Click on a tournament to add entries, then record pool and direct elimination results.

  8. View Analytics

    Check out the analytics section to see performance trends and statistics.
